# Messaging App

A real-time messaging application built with **React** (frontend) and **Node.js/Express** (backend) with **PostgreSQL** for the database. Users can register, login, send messages, join groups, and see online status of other users.

---

## Demo
https://messaging-app-frontend-k9y4.onrender.com

## Features

- User authentication (register, login, logout)
- Real-time user online status
- Direct messages between users
- Group messaging
- Clean, responsive UI with React
- Secure password hashing with bcrypt
- JWT-based authentication

---

## Tech Stack

- **Frontend:** React, React Router, Context API
- **Backend:** Node.js, Express
- **Database:** PostgreSQL
- **Authentication:** JWT, bcrypt
- **Dev Tools:** Nodemon, dotenv, CORS, cookie-parser
